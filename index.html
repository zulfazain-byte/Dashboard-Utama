Pusat Bantuan & Panduan

Pertanyaan Umum (FAQ)

📘 Bagaimana cara menambah stok baru?

Buka tab Stok & Batch, isi form Tambah Batch Baru dengan:

  • Pilih produk dari dropdown
  • Masukkan berat kotor batch (kg)
  • Masukkan harga beli per kg dari supplier
  • Isi ongkir, bensin, dan bongkar muat jika ada
  • Pilih tanggal produksi dan kadaluarsa
  • Klik Simpan Batch
Batch baru akan langsung masuk ke sistem dan stok produk bertambah.

📗 Apa perbedaan FEFO dan FIFO?

FEFO (First Expired First Out) – Batch dengan tanggal kadaluarsa terdekat akan dijual lebih dulu. Cocok untuk produk frozen yang sensitif terhadap masa simpan.
FIFO (First In First Out) – Batch yang masuk pertama akan dikeluarkan lebih dulu. Cocok untuk produk non‑perishable.
Anda bisa mengubah metode ini di tab Pengaturan.

📙 Bagaimana cara backup dan restore data?

Klik tombol Backup di pojok kanan atas → file JSON akan terunduh.
Untuk memulihkan, klik Restore → pilih file backup → data akan dikembalikan dan halaman dimuat ulang.
Tips: Backup setiap minggu untuk keamanan.

📕 Apa itu HPP dan bagaimana cara menghitungnya?

HPP (Harga Pokok Penjualan) adalah total biaya yang dikeluarkan untuk memperoleh produk per kg. Sistem menghitungnya dari:
(Harga Beli × Berat) + Ongkir + Bensin + Bongkar dibagi Berat.
Anda bisa melihat HPP setiap batch di modal Detail Batch.

📒 Bagaimana mengatur harga jual online vs offline?

Buka tab Harga Jual (Online/Offline). Di sana Anda bisa menentukan harga per kg untuk masing‑masing channel. Harga online biasanya lebih tinggi untuk menutupi komisi marketplace. Jika dikosongkan, sistem otomatis menghitung dari HPP + margin.

📓 Bagaimana cara melakukan Stock Opname?

Buka tab Stock Opname, pilih produk, masukkan jumlah fisik yang ada di gudang (hasil timbangan), dan sistem akan mencatat selisih dengan stok sistem. Lakukan ini secara berkala untuk menjaga akurasi data.

Panduan Modul Utama

Stok & Batch

  • Tambah batch baru lengkap dengan landed cost.
  • Lihat ringkasan stok per produk dan status (aman/menipis/kosong).
  • Detail batch: edit, hapus, dan pantau HPP.
  • Batch terpakai hari ini ditampilkan otomatis.

Penjualan

  • Catat penjualan offline (toko) dan online (marketplace).
  • Preview harga sebelum memproses transaksi.
  • Invoice PDF profesional dengan DPP & PPN.
  • Bagikan invoice via WhatsApp langsung.

Keuangan

  • Ringkasan pendapatan, HPP, laba kotor, dan laba bersih bulan berjalan.
  • Catat beban operasional (listrik, sewa, gaji, dll).
  • Buku besar (jurnal) dengan ekspor Excel dan PDF.
  • Grafik laba rugi tahunan.

Laporan

  • Laba rugi detail, neraca, dan stok per gudang.
  • Penjualan per channel, top pelanggan, retur.
  • Proyeksi keuangan 3 bulan ke depan.
  • Grafik tren penjualan bulanan.

Tips & Trik

🚀 Efisiensi Operasional

  • Gunakan shortcut Ctrl+D (Dashboard), Ctrl+S (Stok), Ctrl+B (Backup).
  • Setel target penjualan bulanan di Pengaturan untuk memantau progres di dashboard.
  • Manfaatkan widget toggle di dashboard untuk menampilkan hanya informasi yang Anda butuhkan.

💰 Margin & Harga

  • Tetapkan margin default di Pengaturan, lalu sesuaikan harga online/offline secara manual jika perlu.
  • Gunakan fitur Preview Harga sebelum menjual untuk melihat estimasi laba.
  • Pantau laporan Margin per Produk untuk mengidentifikasi produk paling menguntungkan.

📦 Manajemen Stok

  • Lakukan stock opname rutin untuk mendeteksi selisih sejak dini.
  • Perhatikan indikator warna di tabel stok: 🔴 Kosong, 🟡 Menipis, 🟢 Aman.
  • Batch hampir kadaluarsa akan muncul di dashboard agar segera diprioritaskan.

📊 Pelaporan

  • Gunakan filter tanggal di tab Laporan untuk melihat periode tertentu.
  • Ekspor laporan ke Excel atau PDF untuk keperluan akuntansi atau pajak.
  • Audit Trail mencatat semua perubahan data – berguna untuk investigasi kesalahan.

Pemecahan Masalah Umum

❌ Tombol di sidebar tidak bisa diklik

Lakukan hard reload (Ctrl+Shift+R) untuk membersihkan cache. Jika masih bermasalah, periksa Console browser (F12) – biasanya ada file JavaScript yang gagal dimuat. Pastikan semua file .js di folder js/ tersedia.

💾 Data tidak tersimpan setelah refresh

Data disimpan di browser menggunakan localforage (IndexedDB). Pastikan browser tidak dalam mode Private/Incognito. Jika penyimpanan browser penuh, hapus data aplikasi lain atau lakukan backup manual.

🖨️ Invoice PDF tidak muncul

Pastikan pop‑up tidak diblokir. Browser mungkin meminta izin untuk mengunduh file. Coba nonaktifkan pemblokir pop‑up sementara.

📈 Grafik tidak tampil

Grafik menggunakan Chart.js yang dimuat dari CDN. Periksa koneksi internet. Jika offline, grafik tidak akan dirender, tetapi data tetap bisa diakses melalui tabel.
